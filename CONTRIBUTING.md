# Contributing

Thanks for your interest in FinoCurve App.

At the moment, this repository is being opened first to provide transparency and trust around the desktop client. External contributions may be accepted later as the public workflow matures.

## For now

- Feel free to open issues for bugs, UX problems, or documentation gaps
- For larger feature work, please open an issue first before starting implementation
- Security-sensitive issues should not be disclosed publicly in an issue thread

## Scope reminder

This repository contains the desktop/web client surface of FinoCurve.
It does not include the full managed backend or commercial aggregation stack.

## Local development

```bash
npm install
npm run dev
```

## Style

- Prefer small, focused pull requests
- Keep documentation in sync with code changes
- Avoid introducing hardcoded credentials, tokens, or machine-specific paths
